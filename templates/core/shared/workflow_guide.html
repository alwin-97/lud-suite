Workflow Guide

This guide walks through the recommended setup and day-to-day flow for admins, mentors, and mentees.

Admin Setup
  1. Create mentor, mentee, endorser, and reviewer users.
  2. Assign mentors to mentees and endorsers to mentors.
  3. Configure statuses, session types, and rating domains.
  4. Review the dashboard to confirm assignments and counts.
Admin Config Tips
  • Statuses: Not Started, In Progress, Completed, On Hold.
  • Session Types: Summer Camp, Follow Up, Review.
  • Rating Domains: Leadership, Communication, Technical Skills.
  • Template Configs: Year 1, Year 2, Year 3, Year 4.
Mentor Flow
  1. Open assigned mentees from the Mentor Dashboard.
  2. Review objectives, year plans, and assessments.
  3. Leave comments and update review status when needed.
  4. Export progress reports for review meetings.
Mentee Flow
  1. Update objectives with progress and evidence notes.
  2. Update year plan milestones and remarks.
  3. Check mentor comments and status updates.
  4. Export progress for personal records.
Assessment Flow
  1. Mentors create assessments for each session type.
  2. Select ratings for each configured domain.
  3. Review averages and adjust comments if needed.
  4. Mentees can view assessment history anytime.
Activity Log Flow
  1. Mentors add activities in the YCLP area.
  2. Endorsers review mentor activities and add remarks.
  3. Use exports to capture activity summaries.
