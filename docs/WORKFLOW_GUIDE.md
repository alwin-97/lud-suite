# Workflow Guide

This guide explains the recommended setup and daily flow for admins, mentors, mentees, and endorsers.

## Admin Setup

1. Create users (mentor, mentee, endorser, reviewer) in Manage Users.
2. Assign mentors to mentees.
3. Assign endorsers to mentors (if required).
4. Configure statuses, session types, rating domains, and template configs.
5. Review dashboards to confirm counts and assignments.

## Admin Config Tips

- Statuses: Not Started, In Progress, Completed, On Hold, Delayed
- Session Types: Summer Camp, Follow Up, Review
- Rating Domains: Leadership, Communication, Technical Skills
- Domain Indicators: Attendance, Participation, Submission Quality
- Mood Categories: Motivated, Neutral, Stressed
- Reference Content: Handbook PDF, Evaluation Rubric, Policy Sheet
- Template Configs: Year 1, Year 2, Year 3, Year 4

## Mentor Flow

1. Open assigned mentees from the Mentor Dashboard.
2. Review objectives, year plans, and assessments.
3. Leave mentor comments and update review status.
4. Export progress reports for review sessions.

## Mentee Flow

1. Update objectives with progress and evidence notes.
2. Update year plans with milestone status and remarks.
3. Check mentor comments and status updates.
4. Export progress for personal tracking.

## Assessment Flow

1. Mentors create assessments for each session type.
2. Select ratings for each configured domain.
3. Review averages and adjust notes if needed.
4. Mentees can review assessment history at any time.

## Activity Log Flow (YCLP)

1. Mentors add activities in the YCLP module.
2. Endorsers review mentor activities and add remarks.
3. Export activity logs when needed.
